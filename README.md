# OmniStock

OmniStock is a full-stack E-Commerce and Inventory Management System (ERP).

## Features
- E-commerce / Catalog functionality
- Cart & Checkout System
- Full Admin Dashboard (Inventory, Order Fulfillment, Audit Logs)
- JWT Authentication (Admin & Customer roles)

## Tech Stack
- Frontend: React (Zustand, Tailwind CSS)
- Backend: .NET 8, Entity Framework Core
- Database: PostgreSQL
